Connecting to Backend

To connect to the JadeAssist backend API, you'll need to:

  1. Deploy the backend API server (see packages/backend)
  2. Ensure the POST /api/chat endpoint is accessible
  3. Configure CORS to allow requests from your domain
  4. Initialize the widget with apiBaseUrl pointing to your API

See packages/widget/README.md for detailed deployment instructions.

Features & Persistence

The widget automatically saves your conversation to localStorage. Try:

  • Start a conversation with the widget
  • Refresh the page or navigate to another page
  • The conversation will persist and you can continue where you left off
  • Widget open/closed state is also remembered
